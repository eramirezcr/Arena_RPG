# Arena RPG: El Despertar del Gladiador

Bienvenido a Arena RPG, un juego de combate táctico por turnos desarrollado en Python. Enfréntate a enemigos formidables, gestiona tus estadísticas y sobrevive a las oleadas de combate en la arena definitiva.

# La Leyenda de Aethelgard (Historia)
En las tierras olvidadas de Aethelgard, existe la Gran Arena de Hierro, un lugar donde el tiempo parece detenerse y solo el acero tiene la palabra. No llegaste aquí por gloria, sino por destino. Como un guerrero sin pasado, has sido arrojado a la arena para entretener a antiguos dioses que se alimentan del valor de los mortales.
Cada victoria te otorga fragmentos de esencia que te permiten fortalecer tu cuerpo y mente, pero ten cuidado: los enemigos se vuelven más oscuros y poderosos con cada amanecer. ¿Lograrás alcanzar la libertad o te convertirás en un eco más en los muros de la arena?

# ¿De qué trata el juego?

## Arena RPG 
Es una experiencia de rol centrada en el combate estratégico. El núcleo del juego se basa en:
Combate por turnos: Cada movimiento cuenta. Decide cuándo atacar, cuándo defenderte o cuándo usar habilidades especiales.
Sistema de Progresión: A medida que derrotas enemigos, ganas experiencia para mejorar atributos clave como:
Fuerza: Para golpes demoledores.
Agilidad: Para esquivar y atacar primero.
Magia/Energía: Para desatar poderes ocultos.
Enemigos Escalables: La dificultad aumenta, obligándote a optimizar tu build de personaje para sobrevivir.

# Cómo Jugar

1 Necesitas tener instalado la máquina virtual de Java (JVM), Git y algún ide de desarrollo que soporte Java.

2 Clona el repositorio https://github.com/eramirezcr/Arena_RPG.git

3 Corre el juego, usa la consola para ver las opciones de juego y el teclado para atacar y defenderse de tu oponente.
